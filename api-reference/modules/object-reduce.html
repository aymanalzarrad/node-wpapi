object-reduce Module

Defined in: lib/util/object-reduce.js:3

Utility method to permit Array#reduce-like operations over objects

This is likely to be slightly more inefficient than using lodash.reduce, but results in ~50kb less size in the resulting bundled code before minification and ~12kb of savings with minification.

Unlike lodash.reduce(), the iterator and initial value properties are NOT optional: this is done to simplify the code, this module is not intended to be a full replacement for lodash.reduce and instead prioritizes simplicity for a specific common case.
